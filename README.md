# Credit Risk Analysis

## Overview

This project presents an end-to-end credit risk analysis using Python, SQL, and Power BI. The objective is to identify the key factors influencing loan defaults through data cleaning, exploratory data analysis, SQL-based business analysis, and interactive dashboard development.

## Key Results

- Overall loan default rate: **21.87%**
- Younger borrowers exhibited higher default rates.
- Lower-income borrowers were more likely to default.
- Previous default history strongly influenced future defaults.
- Debt Consolidation loans recorded the highest default rate.
- Lower loan grades were associated with greater credit risk.
